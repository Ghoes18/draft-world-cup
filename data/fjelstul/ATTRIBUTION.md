# Fjelstul World Cup Database (cached CSVs)

CSV files in this folder are downloaded automatically by:

```bash
pnpm build:catalog
```

**Source:** [Fjelstul World Cup Database](https://github.com/jfjelstul/worldcup) by Joshua C. Fjelstul  
**License:** [CC-BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)

You must retain attribution when redistributing derived data. Player `force` values
generated from this data are **autoral estimates** (appearances + goals), not
live 7a0 ratings.
